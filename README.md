Hospital Queue Management System

A modern, user-friendly Hospital Queue Management System designed to reduce waiting time, streamline patient flow, and improve hospital efficiency through real-time queue tracking.

Overview

This project is a web-based application that allows patients to:

Find nearby hospitals
View available doctors
Join queues digitally
Track their position in real-time

Doctors can:

Manage their queue
Call next patients
Pause/resume queue
Track daily consultations
✨ Features
👩‍⚕️ Patient Side
🔍 Search hospitals and departments
👨‍⚕️ View doctors with availability status
🎟️ Generate token and join queue
⏱️ Real-time wait time estimation
📊 Track queue position and progress
❌ Leave queue anytime
🩺 Doctor Side
📋 View and manage patient queue
▶️ Call next patient
⏸️ Pause / Resume queue
📈 Track number of patients served
🔄 Real-time updates
🛠️ Tech Stack
Frontend: HTML, CSS (Tailwind CSS), JavaScript
UI Design: Responsive & modern interface
Logic: Vanilla JavaScript (no backend required for demo)
